# Product

## Register

product

## Users

Individual developers, students, small teams, and professional security engineers who need to evaluate automated code repair or identify AI-specific security risks in source repositories and public websites. They may be learning, reviewing a pull request, preparing a release, or triaging an exposed application.

## Product Purpose

Build one portfolio-grade, open-source monorepo with two complementary tools: FreshPatch, a reproducible benchmark generator and execution-based evaluator for code-repair systems; and Sentinel, a defensive AI security checker that scans local source code and public website surfaces. Success means a new user can run a meaningful local demonstration quickly, understand every finding, reproduce the benchmark, and export evidence suitable for engineering review.

## Brand Personality

Precise, calm, trustworthy. The product should feel technically serious without becoming intimidating, and should communicate uncertainty honestly.

## Anti-references

Avoid neon “hacker movie” visuals, fear-based breach language, opaque risk scores, noisy enterprise dashboards, generic SaaS card grids, and decorative AI imagery. Do not imply that automated scanning replaces a professional penetration test.

## Design Principles

1. Evidence before alarm: every result must show what was observed, why it matters, and how to verify it.
2. Progressive depth: make the first run approachable while preserving detailed technical context for experts.
3. Safe by construction: default to non-destructive, bounded checks and make active network behavior explicit.
4. Honest uncertainty: distinguish confirmed evidence, heuristics, and unavailable checks.
5. Reproducibility is a feature: commands, inputs, environments, and outputs should be easy to repeat.

## Accessibility & Inclusion

Target WCAG 2.2 AA. Support keyboard navigation, visible focus, reduced motion, semantic headings and landmarks, screen-reader labels, and color-independent severity indicators. Use plain language alongside security terminology and never rely on color alone to communicate status.
